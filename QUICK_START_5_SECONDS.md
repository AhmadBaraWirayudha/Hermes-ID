# Quick Start

## Windows

Double-click:

```text
START_INDOMARKET_WINDOWS.cmd
```

Then open:

```text
http://localhost:8501
```

## If it does not work

Double-click:

```text
CHECK_AND_FIX_WINDOWS.cmd
```
