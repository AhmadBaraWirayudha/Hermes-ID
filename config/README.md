# Configuration

Configuration templates and runtime settings.

Important files:

- `settings.example.json` - app settings template
- `alerts.example.json` - alert rule template
- `sources.example.json` - scraper source template
- `rbac.example.json` - roles/permissions template
- `users.example.json` - local user template
- `indonesian_sentiment_lexicon.json` - simple sentiment lexicon

Never commit real secrets. Use environment variables or secret managers.
