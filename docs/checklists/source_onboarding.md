# Source Onboarding Checklist

- [ ] Source owner identified.
- [ ] Terms of service reviewed.
- [ ] robots.txt checked where applicable.
- [ ] Rate limit agreed.
- [ ] Source type selected.
- [ ] Test scrape completed.
- [ ] Normalized columns verified.
- [ ] Data quality reviewed.
- [ ] Alert thresholds configured.
- [ ] Source set active only after validation.
