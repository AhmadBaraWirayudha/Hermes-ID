# Friendly Workflow

## Daily use

1. Start the app.
2. Check Overview.
3. Check Live Monitor.
4. Review Alerts.
5. Open Charts.
6. Run Forecast if needed.
7. Download a report.

## Add data

Use one of these:

- Import CSV
- Sources
- Google Trends
- Live Monitor

## When something looks wrong

1. Open Checkup.
2. Run `CHECK_AND_FIX_WINDOWS.cmd` on Windows.
3. Run `python scripts/doctor.py --fix` on any system.
4. Create a debug bundle with `python scripts/create_debug_bundle.py`.
