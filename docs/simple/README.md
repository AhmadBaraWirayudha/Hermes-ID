# Simple Guide

This folder explains the project in plain words.

## What is Hermes Analytics ID

Hermes Analytics ID is a local market dashboard for Indonesia.

It helps you:

- collect market data
- watch live news and alerts
- read simple charts
- forecast prices
- check data quality
- make reports
- save files as CSV, Excel, PDF, and TeX

## Best first steps

1. Open the app.
2. Click Generate demo data.
3. Go to Overview.
4. Go to Live Monitor.
5. Go to Downloads to make a report.

## New user path

Start with the Start page. It gives you four buttons:

- Add demo data
- Check app health
- Seed live sources
- Seed watchlist

Then open Overview and Live Monitor.
