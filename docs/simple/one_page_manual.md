# One Page Manual

## What this app is

Hermes Analytics ID is a friendly local dashboard for Indonesian market data.

It helps you:

- add data
- watch live signals
- read charts
- check data health
- forecast prices
- make reports

## Best order to use it

1. Start
2. Add demo data or import CSV
3. Open Overview
4. Open Live Monitor
5. Check Alerts
6. Open Charts
7. Run Forecast
8. Download a report

## If you only have 30 seconds

1. Click Start.
2. Click Add demo data.
3. Open Overview.
4. Open Downloads.

## If something breaks

Windows:

```text
CHECK_AND_FIX_WINDOWS.cmd
```

Any system:

```bash
python scripts/doctor.py --fix
```

## Main files to remember
