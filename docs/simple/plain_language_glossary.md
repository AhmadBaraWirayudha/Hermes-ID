# Plain Language Glossary

| Word | Simple meaning |
|---|---|
| API | A way for apps to talk to each other |
| CSV | A simple spreadsheet file |
| Database | A place where data is saved |
| Forecast | A best guess about what may happen next |
| Backtest | A test that checks if a forecast worked in the past |
| Alert | A warning message |
| Signal | A clue that something important may be happening |
| Source | A place where data comes from |
| Scraper | A tool that reads allowed data from a web page |
| RSS | A simple news feed |
| Watchlist | A list of words or topics to watch |
| Sentiment | Whether text sounds positive, negative, or neutral |
| Portfolio | A group of investments |
| Scenario | A what if situation |
| Backup | A saved copy for safety |
| Recovery | Restoring the app after a problem |
| Podman | A free tool to run app containers |
| Local | Running on your own computer |
| Cloud | Running on an online server |
