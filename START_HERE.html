IndoMarket Insight

Use the one-click launcher files in this folder. Browsers cannot directly run local scripts for security reasons, so double-click the file in File Explorer.

Windows

Double-click START_INDOMARKET_WINDOWS.cmd.

For dashboard plus API, double-click START_INDOMARKET_WITH_API_WINDOWS.cmd.

To stop, double-click STOP_INDOMARKET_WINDOWS.cmd.

Linux

Run ./START_INDOMARKET_LINUX.sh.
